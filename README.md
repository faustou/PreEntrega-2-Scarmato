## INSTALLATION

Clonar el repositorio e instalar las dependencias /
Clone the repository and install the dependencies /

```sh
$ git clone https://github.com/faustou/PreEntrega-2-Scarmato
$ cd my-app
$ npm install
```

Iniciar la aplicación / Start the application

```sh
$ npm run dev
```
